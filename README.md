# 🎵 Kanon

> An exploration of canonical specification formats for AI-generated systems

## Why ?

A [canon](https://en.wikipedia.org/wiki/Canon_(music)), in music, is a form where multiple voices sing the same melody, each entering at a different time, producing harmony through their offset.

The same idea can apply to software: a single intention, sung in two voices. The human writes the spec, the machine writes the code. Same theme, different registers, working together towards something coherent.

The spec is canonical because it is the shared melody both voices follow.
